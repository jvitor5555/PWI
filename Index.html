Parágrafos e Formatações de Textos

Este é um páragrafo básico. O elemento <p> é usado para criar páragrafos em HTML.

Texto em negrito pode ser criado com o elemento <strong> ou <b>.

Texto em Itálico pode ser criado com o elemento <em> ou <i>.

Esta linha horizontal foi criada com o elemento <hr>, que é usado para separar seções de conteúdo.
